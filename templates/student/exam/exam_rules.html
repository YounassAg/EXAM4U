Règles de l'examen

Avant de commencer l'examen {{ exam.title }}, veuillez lire attentivement les règles suivantes.

Actions surveillées pendant l'examen

{% include 'components/icons.html' with icon="alert-triangle" class="w-5 h-5 text-red-500 dark:text-red-400" %}

Navigation

Quitter la page d'examen ou changer d'onglet sera enregistré comme une tentative de fraude.

{% include 'components/icons.html' with icon="copy" class="w-5 h-5 text-red-500 dark:text-red-400" %}

Copier-Coller

Toute tentative de copier du texte sera détectée et enregistrée.

{% include 'components/icons.html' with icon="keyboard" class="w-5 h-5 text-red-500 dark:text-red-400" %}

Raccourcis clavier

L'utilisation de raccourcis suspects (Ctrl+C, Ctrl+V, etc.) sera enregistrée.

{% include 'components/icons.html' with icon="monitor" class="w-5 h-5 text-red-500 dark:text-red-400" %}

Redimensionnement de l'écran

Le redimensionnement de la fenêtre sera détecté et enregistré.

{% include 'components/icons.html' with icon="wifi-off" class="w-5 h-5 text-red-500 dark:text-red-400" %}

Connexion Internet

Les déconnexions suspectes seront enregistrées. Une déconnexion prolongée peut entraîner l'arrêt de l'examen.

{% include 'components/icons.html' with icon="alert-circle" class="w-5 h-5 text-red-500 dark:text-red-400" %}

Toutes ces actions sont enregistrées et peuvent être consultées par votre enseignant.
